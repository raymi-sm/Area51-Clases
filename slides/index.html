Introducción a CSS

¿Qué es?

CSS es un lenguage que, a través de una colección de propiedades, se encarga de la capa de presentación de un sitio web.

¿Cómo funciona CSS?

La clave para entender como funciona CSS es imaginar que existe una caja invisible dentro de cualquier elemento HTML

Reglas CSS

Selectores: indican a qué elemento se aplica la regla.

Declaraciones: indican cómo deben estilizarse los elementos mencionados en el selector.

Propiedades: indican los aspectos del elemento que se quiere cambiar.

Valores: especifican la configuración que desea utilizar para las propiedades elegidas.

Reglas CSS y la Cascada

Cuando más de una regla es aplicada al mismo elemento, es importante entender cual de las reglas va a tomar precedencia

  • La última regla, gana: Se aplica en casos en donde los selectores son idénticos.
  • Especificación: Para el caso en donde un selector es más específico que los demás
  • !important: Para el caso en que se quiera forzar una regla a que tome precedencia
