Pseudo Elementos y Clases

Pseudo Elementos: Elementos "fantasma" que pueden ser usados como elementos HTML normales. Es decir, estos elementos no existen dentro del arbol del DOM pero podemos hacerlos aparecer via CSS

  • :after
  • :before
  • :first-letter
  • :first-line

Pseudo Clases: Estado "fantasma" de un elemento HTML que puede ser digido con CSS. Ejemplos de pseudo clases:

  • :link
  • :visited
  • :hover
  • :first-child

Selectores de Atributos

Los selectores de atributo existen para crear reglas CSS a aquellos elementos que tienen un mismo valor de atributo

Selector Significado Ejemplo
Existencia [] Coincide con cualquier atributo sin importar su valor p[class]
Igualdad [=] Coincide con cualquier atributo con un valor especifico p[class="perro"]
