Estructura

Web

Revista

De la misma manera en que un documento de Word contiene encabezados, HTML también se encarga de estructurar la información en los sitios web. Sea para encabezados, párrafos, citas, etc. Todo esto lo hace a través de las etiquetas HTML

Etiquetas HTML

¿Que son?

Las etiquetas de HTML nos ayudan a decirle al navegador que tipo de contenido deseamos mostrar

Sintaxis

<p>Aquí va texto</p>

En la mayoría de los casos la sintaxis de las etiquetas consta de una etiqueta de apertura y otra de cierre en donde en el medio irá el texto que queremos mostrar.

<img />

De igual manera existen casos en donde las etiquetas no necesitan una etiqueta de cierre, debido a que la información que se desea mostrar va dentro de un atributo.

Atributos

¿Que son?

<p id="parrafo">Parrafo con ID</p>

Los atributos son información adicional que se le puede dar a las etiquetas. Los atributos aparecen en la etiqueta de apertura y su sintaxis está armada en dos partes: un nombre y un valor, separados por un signo =

Mi primer sitio

  • <html>: Esta es la etiqueta que guarda todo el contenido de nuestro archivo HTML.
  • <head>: Dentro de esta etiqueta se guarda toda información que el usuario no ve. Información sólo para el navegador.
  • <title>: Aquí va el nombre del sitio web que irá impreso en el la parte superior de la ventana del navegador.
  • <body>: Dentro de esta etiqueta va toda la información que el usuario verá en su pantalla.

Recordar

  • 1. Un sitio web hecho en HTML son documentos de texto
  • 2. HTML utiliza etiquetas que actúan como contenedores y dan información sobre el contenido que está dentro de estas
  • 3. Las etiquetas también son conocidas como elementos
  • 4. Usualmente las etiquetas vienen en pares. La etiqueta de apertura denota el cominzo del contenido mientras que la de cierre denota el final.
  • 5. Las etiquetas de apertura pueden llevar atributos, que nos dan mayor información sobre el contenido de un elemento
  • 6. Los atributos necesitan un nombre y un valor
