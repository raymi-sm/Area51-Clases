Recordar

  • La etiqueta <table> se utiliza para agregar tables a nuestro sitio
  • Las tablas son armadas a partir de filas. Cada fila es creada con la etiqueta <tr>
  • Dentro de cada fila se pueden agregar un número infinito de celdas, representadas por la etiqueta <td> ( o <th> si es un encabezado)
  • Podemos agrupar las tables que son muy grandes en<thread> <tbody> <tfoot>

Formularios

Un formulario consiste en un grupo de elementos que nos permiten recolectar información de las personas que visitan nuestro sitio web.

Como crear un formulario en tu sitio

Entradas de Texto

  • <input type="text">
  • <input type="password">
  • <textarea name="comments"></textarea>

Tomando Decisiones

  • <input type="radio">Rock<input type="radio">Pop<input type="radio">Classic
    Rock Pop Classic 
  • <input type="checkbox">Yellow<input type="checkbox">Red<input type="checkbox">Blue
    Yellow Red Blue

Enviar Información

  • <input type="image" src="ruta/de/la/imagen.png" width="100">
  • <input type="submit" name="submit-btn" value="Enviar">
