JavaScript: Introducción

¿Qué es?

JavaScript es un lenguage de programación utilizado para darle funcionalidad a un sitio web. Puede ir desde efectos visuales hasta aplicaciones complejas como Google Docs y Netflix

Conectar JavaScript con HTML

Existen 2 formas de enlazar nuestro código JavaScript al HTML

  • Archivo Externo
  • Dentro del mismo HTML

Nota: Existe una forma en que podemos escribir instrucciones en JavaScript directamente al navegador y es usando la consola de Chrome

Depuración de Errores (Debugging)

Sintaxis

Usar un lenguage de programación significa dar instrucciones. En este caso van hacia el navegador. Cada lenguage programación consta de su vocabulario y gramática. Más conocido como sintaxis.

alert('mensaje');

El ejemplo de arriba muestra una simple instrucción que se le hace al navegador pidiendo que mande un mensaje.

Escribiendo tu primer programa

alert('Mi primer programa en JavaScript');

Lo que está escrito arriba se le conoce como una declaración de javascript vendría a ser el equivalente a una oración. Así como una oración termina con un punto, una declaración de javascript termina en un punto y coma.

Recordar

  • Colocar nuestro archivo de JavaScript al término de nuestro HTML. Justo antes del cierre de la etiqueta <body>
  • Un programa es un conjunto de múltiples declaraciones en JavaScript que corren una luego de la otra.
  • Una declaración necesita estar terminada para que la siguiente declaración corra.

Variables: Almacenamiento y seguimiento de información

Una variable es como una caja donde puedes colocar información, mirar dentro de la caja para ver que es lo que hay, vaciar la caja e incluso cambiar lo que había antes por algo nuevo.

Mientras que el contenido de la caja cambia, la caja como tal sigue siendo la misma.
